## Descripción
Este proyecto simula un semáforo utilizando **Workers** en JavaScript para manejar la computación paralela. Cada semáforo cambia de color de forma coordinada, asegurando que solo una dirección esté en verde mientras las otras permanecen en rojo, replicando el comportamiento de un cruce real.

## Tecnologías
- React con vite
